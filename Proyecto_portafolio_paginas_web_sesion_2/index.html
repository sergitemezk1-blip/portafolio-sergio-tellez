Planificación y administración de redes.

Proyecto realizado para la asignatura Planificación y Administración de Redes. El trabajo consistió en el diseño, configuración y verificación de una red pequeña y funcional, utilizando direccionamiento IPv4 y el simulador Cisco Packet Tracer. Durante el desarrollo del proyecto se realizaron las siguientes tareas:

  • Diseño de un plan de direccionamiento IPv4, partiendo de la red 192.168.1.0 y aplicando subnetting para crear 8 subredes.
  • Configuración de una topología de red con router, switches y PCs, asignando direcciones IP estáticas a cada dispositivo.
  • Activación y configuración de las interfaces del router para permitir la comunicación entre subredes.
  • Verificación del funcionamiento de la red mediante pruebas de conectividad (ping) entre hosts y hacia las puertas de enlace.
  • Documentación del proceso, incluyendo tablas de direccionamiento, diagramas y evidencias de las pruebas realizadas.

Este proyecto permitió aplicar de forma práctica los conceptos básicos de redes, comprender el uso del subnetting y validar la correcta comunicación entre diferentes segmentos de red.

Fundamentos de hardware.

Proyecto realizado para la asignatura Fundamentos de Hardware, centrado en el análisis práctico de los componentes de un sistema informático mediante el uso de máquinas virtuales. El trabajo aborda la identificación del hardware, pruebas de rendimiento, verificación del estado del almacenamiento y gestión de discos, utilizando herramientas reales como CPU-Z, CrystalDiskMark, GParted y entornos virtualizados para simular escenarios profesionales. Principales aprendizajes:

  • Análisis y verificación de componentes hardware.
  • Evaluación del rendimiento del sistema.
  • Gestión básica de discos y particiones.
  • Uso de máquinas virtuales como entorno seguro de pruebas.
  • Comprensión de procesos habituales en soporte técnico y administración de sistemas.
